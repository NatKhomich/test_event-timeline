## Установка и запуск

1. Установите зависимости:

   ```
   npm install
   ```

2. Запустите проект в режиме разработки:

   ```
   npm start
   ```

   Откройте [http://localhost:3000](http://localhost:3000) в браузере.

3. Для сборки production-версии:
   ```
   npm run build
   ```
